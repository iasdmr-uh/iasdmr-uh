# Official Branding Guide

Reference: ES brandbook.pdf (Seventh Day Adventist Reform Movement)

## Colors

| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Primary Blue | `#0A245E` | 10, 36, 94 | Main brand color, navbar, footer |
| Secondary Blue | `#3E80F2` | 62, 128, 242 | Links, highlights |
| Dark Blue | `#0F3999` | 15, 57, 153 | Sidebar, accents |
| Gold Accent | `#D4AF37` | 212, 175, 55 | Buttons, CTAs, highlights |
| Beige | `#EFD19F` | 239, 209, 159 | Secondary accent |
| Coral | `#FF8D6D` | 255, 141, 109 | Alerts, emphasis |
| Gray | `#7E7E82` | 126, 126, 130 | Muted text |
| Off-white | `#F2F1EF` | 242, 241, 239 | Backgrounds |

## Typography

| Usage | Font | Weights |
|-------|------|---------|
| Headings | BW Modelica | Black, ExtraBold, Bold, Medium |
| Body | BW Modelica | Regular, Light |
| Long text | Noto Sans | Regular |

Font files located in: `assets/fonts/`

## Logo Guidelines

- Use only in Primary Blue (#0A245E) or white
- Minimum height: 2cm (use symbol only below this)
- Maintain 1/2x safe area around logo
- Never alter shape, apply shadows, or modify colors outside palette
